# vue-multipage
## 简介：
基于vue-cli3.0 vue多页面项目，超级实用，拿来即用，大大提升开发效率

## 项目功能概述
主要包含以下功能
- filter
- vue-router
- vuex
- 多页面
- 路由懒加载
- 多环境
- gzip压缩
- rem支持
- scss
- vant 有赞UI框架


## 项目页面
初始有以下两个页面,每个页面有各自独立路由（可以根据需要自行添加）
- index 
 可用过 http://localhost:8080/home 访问

- activity
 http://localhost:8080/activity.html#/new-year
